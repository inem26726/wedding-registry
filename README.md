# Wedding Registry CMS

## Features
- List, Add, Edit, Delete Registry Items.
- Secure login for administration.

## Usage
1.  **Access CMS**: Go to `/cms.html` (e.g., `http://localhost:3000/cms.html`).
2.  **Login**: Use your CMS credentials.
3.  **Manage Items**:
    -   **Add**: Click "Tambah Item" button.
    -   **Edit**: Click the edit (pencil) icon on an item.
    -   **Delete**: Click the delete (trash) icon.
4.  **Public View**: The main registry is at `/` (`index.html`).

## Development
-   `server.js`: Backend API (Express + SQLite).
-   `public/cms.html`: Admin Interface.
-   `public/js/cms.js`: Admin Logic.
